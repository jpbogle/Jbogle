Novartis

Interning at Novartis was a great experience where I learned a lot working alongside the companies best informatics software engineers. I was able to start and complete a full stack drawing application that uses a machine learning hidden markov model to detect images that users are drawing. We trained our model on our own data that we collected using a react based data collection UI that we ran on tablets and phones. Then, we used the image recognition to create a react and redux based application for users to rapidly draw shapes and drag them into larger diagrams and protoypes. We also used websockets to allow multiple users to edit the same diagram at the same time.

In my junior winter at MIT, I participated in MIT's web development competition. My teammate and I created our application called Stations and were awarded third place in the competition with over 40 teams. Stations allows users to search through Soundcloud, Spotify, and Apple Music songs and add them to a station. The station is public so other users can go on and recommend, up vote and down vote songs and the station plays on all users that are tuned in. Since then, we have rehauled this web application using a backend written in Go and a react-redux frontend. We also exported it into desktop and mobile applications using electron and react native. We used websockets to connect clients to their station, sessions to keep users logged in, and all Spotify, Soundcloud and Apple Music search APIs. For playback, we used URI schemes to the Spotify and iTunes applications and Soundcloud streaming API. The application is currently hosted on two AWS instances.
